# Customer_shopping_trends_and_Insights
An end-to-end data analytics project that analyzes customer purchasing behavior, spending patterns, and shopping trends to generate actionable business insights using Python and data visualization techniques.

📌 Project Overview

Customer Purchase Pattern Analytics is an end-to-end Data Analytics project that explores customer purchasing behavior using transactional retail data. The project aims to uncover insights into customer demographics, spending habits, product preferences, subscription behavior, and purchasing trends to support data-driven business decisions.

The analysis was performed using Python for data cleaning and exploratory data analysis (EDA), PostgreSQL for business-focused SQL analysis, and Power BI for interactive dashboard development.

🎯 Project Objectives
Analyze customer shopping and spending behavior.
Identify high-value customer segments.
Understand product and category preferences.
Evaluate the impact of discounts and subscriptions on revenue.
Generate actionable business recommendations through data-driven insights.

🔍 Data Cleaning & Preparation

The following preprocessing steps were performed:
Loaded and explored the dataset using Pandas.
Identified and handled missing values.
Imputed missing review ratings using category-wise median values.
Standardized column names using snake_case format.
Created new features such as:
Age Groups
Purchase Frequency Metrics
Removed redundant columns after consistency checks.
Loaded cleaned data into PostgreSQL for advanced SQL analysis.

💡 Key Business Insights
Subscribers contribute significantly to overall revenue.
Loyal customers generate higher long-term value.
Certain products rely heavily on discounts for sales.

🚀 Business Recommendations
